# eduardolagolima.dev

> Meu site pessoal :)

## Tecnologias usadas

- Nuxt.js
- TypeScript
- Bulma
- Netlify

## Como testar

```bash
# clone repository
$ git clone https://github.com/eduardolagolima/eduardolagolima.dev.git

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
